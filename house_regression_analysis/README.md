WEEK 5- ADVANCED MACHINE LEARNING: HOUSE PRICE PREDICTION
AnalystLab Data Science Internship
Overview
This project extends from EDA to a full regression modelling pipeline: baseline Linear Regression, Decision Tree,
Random Forest, and Gradient tuning with Hyper-parameter tuning via GridSearchCV and RandomizedSearchCV.
All models are trained on log1p(price) and predictions were back-transformed using expm1 before scoring. Therefore, 
every metric is real currency units and not log-price units
